Data Cleaning and Visualization Project

Objective
Clean a raw dataset and visualize insights using Python.

Tools Used
* Python
* Pandas
* Matplotlib

Dataset
Titanic Dataset

Tasks Performed
* Handled missing values
* Removed duplicates
* Processed data
* Created visualizations

Visualizations
* Survival Count
* Gender Distribution
* Passenger Class Distribution
* Age Distribution

Key Findings
1. Missing values were cleaned.
2. Most passengers did not survive.
3. Male passengers formed the majority.
4. Most passengers belonged to Class 3.
5. Passenger ages were mainly between 20 and 40 years.
